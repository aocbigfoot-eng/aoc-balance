# 开局平衡器 PWA

这是一个手机网页 App 原型，不需要 App Store 或 Xcode。

## 使用方式

1. 把整个 `rts-balance` 文件夹发布到 HTTPS 静态网站。
2. 用 iPhone Safari 打开发布后的网址。
3. 点 Safari 分享按钮。
4. 选择“添加到主屏幕”。

添加到主屏幕后，它会像 App 一样从桌面启动。

## 当前功能

- 拍照或选图预览
- 手动粘贴/输入 8 个玩家 ID
- 支持 `_5.75` 和 `_Y6` 解析
- 战力按 0.25 档校验
- 自动生成 4v4 最平衡分组
- 复制分组结果
- 发布到 HTTPS 后支持离线缓存

## 当前限制

- OCR 自动识别尚未接入
- 如果直接打开本地 `index.html`，离线缓存不会启用；发布到 HTTPS 后会启用
